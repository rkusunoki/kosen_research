# 進化計算を活用したクレーンシステムの省エネルギー振動制御に関する研究

旭川高専専攻科１年時に実施した研究です，

## 1. 研究背景
本研究では，進化計算を活用したクレーンシステムの省エネルギーフィードフォワード振動制御に関する研究を試みる．具体的には，ケーブル長が変化する天井クレーンのPTP制御問題を扱い，位置決め後の荷揺れを抑制させ，かつ，駆動エネルギーを最小化するトロリーの軌道生成法を提案する.そして，数値シミュレーションから提案手法の有効性を検証する．
